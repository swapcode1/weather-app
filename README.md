## Installation

1. Clone the repository:
\`\`\`bash

2. Install dependencies:
\`\`\`bash
npm install
\`\`\`

\`\`\`
REACT_APP_WEATHER_API_KEY=your_api_key_here

\`\`\`bash
npm start
